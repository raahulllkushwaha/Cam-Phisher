## Reporting a Vulnerability

If you discover a security vulnerability within CamPhish, please follow responsible disclosure practices:

1. **Do NOT** create a public GitHub issue for security vulnerabilities
2. Contact the author directly through GitHub: [@raahulllkushwaha](https://github.com/raahulllkushwaha)
3. Provide detailed information about the vulnerability
4. Allow reasonable time for the issue to be addressed before public disclosure

## Response Timeline

- **Initial Response:** Within 48-72 hours
- **Status Update:** Within 7 days
- **Resolution:** Depends on complexity, typically within 30 days

## Ethical Use Policy

This tool is designed **exclusively** for:
- ✅ Educational purposes
- ✅ Authorized penetration testing
- ✅ Security research in controlled environments

**Unauthorized use is strictly prohibited and may violate local, state, and federal laws.**
